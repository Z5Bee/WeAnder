## Design Choices
- Layout: Designed to create a calm and easy-to-navigate experience. I used the card-based structure to organize features making information easy to scan.
- Color:I followed the brand guidelines for Color and Typography. The color palette uses warm tones inspired by road trips and outdoor exploration.
- Typography: Nunito were used for main text to keep information clear, friendly, and easy to read.
- Visual hierarchy: Through differences in font size and color contrast I expressed visual hierki. Larger headings draw attention first, while supporting text provides additional details.
- Feedback states: Interactive elements include hover effects on cards, buttons, and links to show users that elements are clickable. 
- Responsive design: The design adapts to different screen sizes using flexible layouts and media queries. Cards adjust for smaller screens so the content remains readable and easy to navigate on mobile devices.
